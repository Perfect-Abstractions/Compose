### grantRoleWithExpiry

Grants a role to an account with an expiry timestamp. Only the admin of the role can grant it with expiry. Emits a &#123;RoleGrantedWithExpiry&#125; event. Reverts with AccessControlUnauthorizedAccount If the caller is not the admin of the role.

<ExpandableCode language="solidity" maxLines={8}>
{`function grantRoleWithExpiry(bytes32 _role, address _account, uint256 _expiresAt) ;`}
</ExpandableCode>

**Parameters:**

<PropertyTable
  properties={[
    {
      name: "_role",
      type: "bytes32",
      description: "The role to grant."
    },
    {
      name: "_account",
      type: "address",
      description: "The account to grant the role to."
    },
    {
      name: "_expiresAt",
      type: "uint256",
      description: "The timestamp when the role should expire (must be in the future)."
    }
  ]}
  showRequired={false}
/>
