This module provides internal functions for use in your custom facets. Import it to access shared logic and storage.

## Overview

This module provides internal functions to manage the administrator role for other roles within a diamond. Facets can import this module to set or retrieve role administrator assignments using shared diamond storage. This ensures consistent access control policies across all facets that interact with the same storage.

## Best Practices
