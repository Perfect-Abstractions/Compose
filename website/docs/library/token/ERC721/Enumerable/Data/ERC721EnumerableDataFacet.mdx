<DocSubtitle>
Enumerate ERC-721 tokens by owner and globally
</DocSubtitle>

<Callout type="info" title="Key Features">
- Exposes external view functions for token enumeration.
- Leverages diamond storage pattern for shared state access.
- Provides functions to query total supply, tokens by owner index, and global token index.
- Includes `exportSelectors` for diamond registry discovery.
</Callout>

## Overview

This facet provides external functions for enumerating ERC-721 tokens within a diamond. It accesses shared diamond storage to retrieve token ownership and global token indexes. Developers integrate this facet to enable querying token collections without direct storage access.

## Best Practices

<Callout type="tip" title="Best Practice">
- Ensure this facet is added to the diamond with its correct selectors.
- Verify that the `totalSupply`, `tokenOfOwnerByIndex`, and `tokenByIndex` functions are called through the diamond proxy interface.
- Handle the `ERC721OutOfBoundsIndex` error when querying token indexes.
</Callout>

## Security Considerations

<Callout type="warning" title="Security">
The `tokenByIndex` and `tokenOfOwnerByIndex` functions revert with `ERC721OutOfBoundsIndex` if the provided index is out of bounds relative to the total supply or the owner's token count, respectively. Ensure input validation for `_index` parameters when integrating off-chain.
</Callout>
